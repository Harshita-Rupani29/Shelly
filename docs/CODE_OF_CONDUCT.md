# Code of Conduct

This project adheres to the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct, version 2.0.

For the full text, see [CODE_OF_CONDUCT.md](https://github.com/juspay/shelly/blob/release/CODE_OF_CONDUCT.md) in the repository root.

## Summary

We are committed to providing a welcoming, inclusive, and harassment-free experience for everyone.

## Reporting

Instances of unacceptable behavior may be reported to opensource@juspay.in.

All complaints will be reviewed and investigated promptly and fairly.
