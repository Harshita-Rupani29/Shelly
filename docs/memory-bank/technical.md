# Technical Docs

The `memory-bank/technical/` directory contains architecture and technology documentation.

## Files

### systemPatterns.md

Documents the project's architecture:

- Design patterns in use
- Component relationships
- Data flow patterns
- Key architectural decisions and rationale

### techContext.md

Describes the technology stack:

- Programming languages and versions
- Frameworks and libraries
- Build tools and configuration
- Development environment setup
- External service dependencies

## Generation

Technical documentation is generated by analyzing:

- `package.json` dependencies
- Source file structure and imports
- Configuration files (tsconfig, eslint, etc.)
- Existing documentation

## Updating

```bash
# Update technical docs
shelly memory update

# View current content
shelly memory show technical/systemPatterns.md
shelly memory show technical/techContext.md
```
