# Changelog

All notable changes to Shelly are documented in this file.

This project uses [semantic-release](https://github.com/semantic-release/semantic-release) for automated versioning. The changelog is generated from [Conventional Commits](https://www.conventionalcommits.org/).

For the full release history, see [GitHub Releases](https://github.com/juspay/shelly/releases).

## Recent Highlights

### v1.2.0

- Project scaffolding from `shelly organize`
- Comprehensive neurolink-style templates with organize integration
- Multi-provider AI support (Google AI + Vertex AI)
- Fixed `node-pty` posix_spawnp error on Apple Silicon

### v1.0.0

- Initial release
- Error analysis mode with AI-powered suggestions
- Repository organization command
- Memory Bank management
- GitHub repository setup automation
- Multi-shell support (bash, zsh, tcsh)
